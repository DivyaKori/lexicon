# 🚀 Lexicon – Vocabulary Learning App

Lexicon is a modern and interactive web application designed to help users improve their English vocabulary in a simple and engaging way.

---

## ✨ Features

- 📖 Learn new words with meanings and usage  
- 🧠 Smart vocabulary learning experience  
- 📊 Track your learning progress  
- ⚡ Fast and responsive UI  
- 🎨 Clean and modern design  

---

## 🛠 Tech Stack

- React  
- TypeScript  
- Vite  
- Tailwind CSS
